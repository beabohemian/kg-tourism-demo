# Kyrgyzstan Tourism Digital Platform — Prototype

정부 데모용 단일 파일 프로토타입.
**Министерство туризма Кыргызской Республики** × **Tripsoda × GovTech KG**

🔗 Live: https://beabohemian.github.io/kg-tourism-demo/

## 핵심 3축

1. **Цифровая сертификация гидов** — QR 라이센스 발급 5단계 (Документы → Tundyk МВД → ГНС → ЕЦП → блокчейн)
2. **Контроль налоговых поступлений** — 자동 10% 수취, 분배 시각화, on-chain 원장
3. **Операционное управление** — 지도, 날씨/안전, 매출 추세, 국가별 분석, 캘린더

## 스택

단일 HTML · Tailwind CSS (CDN) · Vue 3 (CDN) · vanilla JS · 모킹 데이터

## 사용

브라우저로 `index.html` 열기. 빌드 없음.
